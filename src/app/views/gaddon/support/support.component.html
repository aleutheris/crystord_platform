Getting Started

What is Crystord?

Crystord is a Google Workspace Add-on that syncs variables between Google Sheets™ and Google Slides™. Use it to create dynamic Slide templates where placeholders like {{ '{{' }}variable_name{{ '}}' }} are replaced with values from your spreadsheet.

Basic setup:

  • Create a table in Sheets with the columns Key and Value, where Key represents the variables names and Value represents the values of the respective variables that you want to insert into Slides.
  • Install the add-on from the Google Workspace Marketplace.
  • Open your Slides presentation and click the Crystord icon in the right-side panel.
  • Copy the Google Sheets™ file ID from your Sheet's URL and paste it into the Crystord add-on sidebar.
  • Add variables as placeholders ({{ '{{' }}variable_name{{ '}}' }}) in your Slides.
  • Sync your data: press the button "Synchronize" to update Slides with the latest values from your Sheet.

Frequently Asked Questions

How do I install the add-on?

Install directly via the Google Workspace Marketplace or from within Google Slides™' side panel by searching for "Crystord".

The add-on can't find my Sheet or gives a permissions error.

  • Make sure the Sheet ID is correct (copy the part between /d/ and /edit in the Sheets URL).
  • Ensure your Google account has at least "Viewer" access to the Sheet.
  • If prompted, approve all requested permissions.

The variables in Slides aren't updating.

  • Double-check that the variable names match exactly (case-sensitive, no extra spaces).
  • Ensure you've clicked "Refresh Variables" after editing values in Google Sheets™.

Do I need to re-link the Sheet every time?

No. Once you've assigned the Sheet ID, it is remembered for the current Slides file. You can refresh values as often as needed.

What format should I use for variables in Slides?

Use curly brackets: {{ '{{' }}variable_name{{ '}}' }} (e.g., {{ '{{' }}item_name{{ '}}' }}).

Troubleshooting

Add-on not appearing?

Refresh Google Slides™ and ensure you are using the same account you installed the add-on with.

Sidebar won't open or is unresponsive?

Try closing and reopening Slides, or reinstall the add-on if needed.

Unexpected errors or bugs?

Take a screenshot or copy the error message and contact support (details below).
