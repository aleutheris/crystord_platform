Description of the Service

The Crystord Add-on allows users to dynamically populate Google Sheets™ with data from Crystord data flows. Cells can be coupled to specific atom features from Crystord, ensuring spreadsheets always reflect the freshest data.

  • Primary functions: Search and couple spreadsheet cells to Crystord atoms, fetch atom data, and synchronise values with granular versioning.
  • Current limitations: The Add-on currently supports text and numeric values and is limited to integration between Crystord and Google Sheets™.
